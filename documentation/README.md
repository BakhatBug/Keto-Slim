## ğŸ“ Project Description

KetoSlim is a comprehensive weight loss application that provides personalized keto diet plans based on user-submitted health data. It features a modern React frontend and a robust Node.js/Express backend with MongoDB.

### Key Features
- **User Authentication**: Secure signup and login functionality.(backend)
- **Personalized Plans**: Generates custom weight loss plans based on BMI, age, and goals.(frontend)
- **E-commerce**: Integrated product catalog and shopping cart(backend only)
- **Order Tracking**: Users can track their order status(backend only)
- **Responsive Design**: Fully responsive UI for mobile and desktop.(frontend)
